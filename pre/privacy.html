2. Dati raccolti

Il nostro sito web può raccogliere esclusivamente dati di navigazione anonimi, come indirizzo IP, tipo di browser, durata della visita e pagine visualizzate. Non raccogliamo attivamente dati personali identificabili.

3. Cookie

Il nostro sito web può utilizzare cookie per migliorare l'esperienza dell'utente. La gestione dei cookie è possibile attraverso le impostazioni del browser.

4. Utilizzo dei dati

I dati anonimi raccolti vengono utilizzati esclusivamente per l'analisi e il miglioramento del sito web. I vostri dati non vengono ceduti a terzi.

5. Siti esterni

Il nostro sito web può contenere link a siti esterni. Non siamo responsabili delle pratiche sulla privacy di tali siti.

6. I vostri diritti

In conformità con le leggi vigenti sulla protezione dei dati (incluso il GDPR), avete il diritto di accedere, rettificare o cancellare i vostri dati.

7. Contatti

Per qualsiasi domanda, vi preghiamo di contattarci attraverso il nostro sito web.
